GIT syntax cheatsheet for reference

cd - Change directory

cd .. - Move out of directory

ls - List all files inside directory

git clone { SOME_URL } - Clone a git repo to the folder you're in.

git remote -v - Check to see if you have a source repository you can push your cahnges to. ( Will explain further. )

git checkout -b { BRANCH_NAME } - Create a new branch for your changes/features/bug fixes etc.

git status - Check your changes ( You will see it in a red color in the command line. )

git add . - Confirm that your changes are ready for pushing.

git commit -m { SOME_MESSAGE } - Add a message that will show up in github that will describe your changes.

git push - Push your changes to the remote branch you set ( SEE git remote -v for this one. Will explain further )

git pull - Pull changes that other people made so you all have the same code moving forward.
